隱私權政策

互動式冒險故事引擎 API 服務

最後更新日期： 2024年7月26日
生效日期： 2024年7月26日

1. 政策概述

本隱私權政策說明我們的互動式冒險故事引擎 API 服務（以下簡稱「本服務」）如何收集、使用、儲存和保護您的資料。本服務專為 ChatGPT 和其他 AI 智能體設計，提供互動式文字冒險遊戲功能。

重要聲明：我們致力於保護您的隱私，僅收集提供服務所必需的最少資料，且不會收集任何個人識別資訊。

2. 服務說明

本服務提供以下功能：

  • 故事引擎 API：載入故事章節、處理遊戲狀態、提供選項分支
  • 擲骰 API：執行隨機數生成和遊戲檢定
  • 條件內容系統：根據遊戲狀態動態顯示內容

3. 資料收集

3.1 我們收集的資料類型

資料類型 收集目的 範例 保存期限
故事章節請求 載入對應的故事內容 chapter_id: 1 請求期間
遊戲狀態變數 處理條件內容和遊戲邏輯 has_key: true, health: 100 請求期間
擲骰參數 執行隨機數生成 dice_sides: 20, dice_count: 1 請求期間
技術日誌 系統維護和錯誤排除 API 呼叫時間、回應狀態 30天

3.2 我們不收集的資料

  • 個人識別資訊（姓名、電子郵件、電話號碼等）
  • IP 地址或裝置識別碼
  • 瀏覽器指紋或追蹤 Cookie
  • 用戶的完整對話記錄
  • 任何敏感個人資料

4. 資料使用

我們收集的資料僅用於以下目的：

4.1 主要用途

  • 提供故事內容：根據章節 ID 回傳對應的故事文字和選項
  • 處理遊戲邏輯：解析條件內容，根據遊戲狀態顯示適當內容
  • 執行擲骰計算：生成隨機數並回傳結果
  • 維護服務品質：監控 API 效能和可用性

4.2 技術用途

  • 錯誤診斷和系統優化
  • API 效能監控
  • 安全性監控和防護

5. 資料保護

5.1 技術保護措施

  • HTTPS 加密：所有 API 通訊都透過 TLS/SSL 加密傳輸
  • 無狀態設計：每次 API 呼叫都是獨立的，不保留用戶狀態
  • 最小權限原則：系統僅具備提供服務所需的最小權限
  • 安全的雲端基礎設施：使用 Render 平台的安全保護

5.2 存取控制

  • 僅授權人員可存取系統日誌
  • 定期更新安全性設定
  • 監控異常存取行為

6. 資料保存和刪除

6.1 資料保存期限

  • 遊戲資料：不保存，每次請求後立即清除
  • 技術日誌：保存 30 天後自動刪除
  • 錯誤日誌：保存 90 天後自動刪除

6.2 自動刪除機制

我們實施自動化的資料清理程序，確保過期資料被及時刪除，無需用戶手動請求。

7. 第三方服務

7.1 雲端服務提供商

  • Render：提供 API 託管服務
  • PostgreSQL：儲存故事內容（不包含用戶資料）

7.2 第三方政策

這些第三方服務都有自己的隱私權政策，我們選擇符合高標準資料保護要求的服務提供商。

8. 用戶權利

8.1 資料透明度

由於我們不收集個人識別資訊，且不保存用戶的遊戲資料，因此沒有需要查詢或刪除的個人資料。

8.2 服務控制

  • 您可以隨時停止使用本服務
  • 您的遊戲狀態完全由 ChatGPT 管理，我們不保存
  • 每次遊戲都是全新開始，沒有歷史記錄

9. 兒童隱私

本服務不專門針對 13 歲以下兒童設計，也不會故意收集兒童的個人資訊。如果我們發現意外收集了兒童的個人資訊，將立即刪除。

10. 政策變更

我們可能會不定期更新本隱私權政策。重大變更時，我們會：

  • 更新本頁面的「最後更新日期」
  • 在適當的情況下提供通知
  • 確保變更符合適用的法律要求

11. 法律依據

我們處理資料的法律依據包括：

  • 合法利益：提供和改善服務
  • 技術必要性：執行您請求的服務
  • 安全需求：保護服務和用戶安全

12. 國際資料傳輸

本服務可能涉及跨國資料傳輸。我們確保所有資料傳輸都符合適用的資料保護法律，並採用適當的保護措施。
